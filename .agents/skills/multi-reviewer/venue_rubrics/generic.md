# Generic Review Rubric

## 适用场景
未指定目标会议/期刊时的默认 rubric，适用于任何学科。

## 评分维度 & 权重

| 维度 | 权重 | 评分说明 |
|------|------|---------|
| **Novelty** | 20% | 研究问题或方法的新颖性 |
| **Soundness** | 25% | 方法论严谨性、实验/论证可靠性 |
| **Significance** | 15% | 对领域和社会的潜在影响 |
| **Clarity** | 15% | 写作质量、逻辑结构、图表清晰度 |
| **Reproducibility** | 10% | 方法描述是否足够详细以复现 |
| **Related Work** | 10% | 文献覆盖广度和准确性 |
| **Ethics & Limitations** | 5% | 局限性讨论、伦理考量 |

## 评分标准 (1-10)

| 分数 | 含义 |
|------|------|
| 9-10 | 杰出 — 原创性极强，论证无懈可击 |
| 7-8 | 优秀 — 有显著贡献，minor issues |
| 5-6 | 一般 — 有贡献但存在明显不足 |
| 3-4 | 较差 — 贡献有限或存在重大问题 |
| 1-2 | 不合格 — 根本性缺陷 |

## 通用审稿要点
1. 研究问题是否明确且有价值？
2. 方法论是否适合回答该研究问题？
3. 数据/实验是否支持结论？
4. 讨论是否客观，是否承认局限性？
5. 写作是否清晰简洁？
