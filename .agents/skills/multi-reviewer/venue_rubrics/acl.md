# ACL Review Rubric

## 页数限制
- Long Paper: 8 页 + 参考文献
- Short Paper: 4 页 + 参考文献
- 附录: 无限制

## 评分维度 & 权重

| 维度 | ACL 专属权重 | 评分说明 |
|------|-------------|---------|
| **Soundness** | 20% | 方法和实验是否正确 |
| **Novelty** | 15% | NLP 领域的方法创新 |
| **Significance** | 15% | 对 NLP/CL 社区的影响 |
| **Clarity** | 15% | 写作质量和可读性 |
| **Reproducibility** | 15% | ACL 特别重视！需详细实验设置 |
| **Related Work** | 10% | NLP 文献覆盖完整性 |
| **Ethics** | 10% | 数据隐私、bias、dual use |

## ACL 特有要求

- **Reproducibility Checklist**: 必填，很详细
- **Responsible NLP Checklist**: 2023 起必填
- **Limitations Section**: 必须有，不写会被扣分
- **Ethics Statement**: 涉及人类数据必须声明 IRB
- **多语言考量**: ACL 偏好多语言评估，仅英语实验可能被质疑

## 评分标准 (1-5)

注意：ACL 用 **1-5 分制**，不同于其他会议。

| 分数 | 含义 |
|------|------|
| 5 | 强烈推荐 — 顶级贡献 |
| 4 | 推荐 — 有显著贡献 |
| 3 | Borderline — 有贡献但有问题 |
| 2 | 不推荐 — 贡献不足 |
| 1 | 强烈不推荐 — 重大缺陷 |

## 常见 Reject 原因

1. 仅在英语上测试，缺少多语言实验
2. 缺少 Limitations Section
3. 数据集存在偏见但未讨论
4. 与 EMNLP/NAACL 的同期工作冲突
5. 人工评估不规范（缺少 IAA、评估者数量不够）
